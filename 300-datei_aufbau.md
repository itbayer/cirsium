# Datei Aufbau

Die Sortierung der Text Daten wird durch eine
Vornummer bestimmt.

Beispiel:

- 000-start.md
- 100-vorwort.md
- 200-makefile.md
- 300-datei_aufbau.md
- 999-ende.md
